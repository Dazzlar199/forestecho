# 네이버 블로그 포스팅 가이드 📝

## 1. 네이버 블로그 개설하기

1. [네이버 블로그](https://blog.naver.com) 접속
2. "블로그 만들기" 클릭
3. 블로그 URL 설정 (예: `blog.naver.com/forestecho_kr`)
4. 블로그 이름: "숲울림 - 심리건강 이야기" 또는 원하는 이름

---

## 2. 포스팅할 콘텐츠

현재 사이트에 있는 **24개의 교육 콘텐츠**를 블로그 포스트로 전환할 수 있습니다:

### 카테고리별 콘텐츠:
- **기본 개념** (2개): 정신건강이란, 정신건강 자가진단
- **정신건강 질환** (5개): 우울증, 불안장애, 공황장애, 사회불안, 강박증
- **관리 및 대처** (4개): 스트레스 관리, 감정 조절, 분노 조절, 슬픔과 상실
- **생활습관** (4개): 수면과 정신건강, 운동, 디지털 웰빙, 자기 돌봄
- **대인관계** (2개): 대인관계 스트레스, 경계선 설정
- **직장/학업** (3개): 직장 스트레스, 번아웃, 완벽주의
- **청소년** (4개): 청소년 우울증, 학업 스트레스, 괴롭힘, 소셜미디어

## 3. 블로그 포스트 작성 방법

### 방법 1: 사이트에서 복사 + 편집 (가장 쉬움!)

1. https://forestecho.app/education 접속
2. 원하는 아티클 클릭
3. 내용 전체 복사 (Cmd/Ctrl + A → Cmd/Ctrl + C)
4. 네이버 블로그 글쓰기 붙여넣기
5. 아래 템플릿 추가:

```markdown
---

## 🌲 숲울림에서 AI 상담 받아보세요

24시간 언제든지, 전문적인 AI 심리상담이 준비되어 있습니다.

👉 [지금 상담하기](https://forestecho.app)

---

#심리상담 #AI상담 #정신건강 #멘탈케어 #우울증 #불안장애 #숲울림
```

### 방법 2: 직접 작성

각 교육 콘텐츠를 자신의 말로 재작성하여 블로그에 포스팅합니다.

---

## 4. 이미지/썸네일 제작하기

### 🎨 Canva 사용 (가장 추천!)

#### A. Canva 시작하기
1. [Canva.com](https://canva.com) 접속 (무료)
2. 회원가입 (구글 계정으로 간편 가입)
3. "블로그 배너" 또는 "인스타그램 포스트" 검색

#### B. 템플릿 활용
1. 검색창에 "mental health", "심리", "health" 입력
2. 마음에 드는 템플릿 클릭
3. 텍스트 수정:
   - 제목: "우울증 극복하는 5가지 방법"
   - 부제: "숲울림 심리교육"
4. 색상 변경: 초록색 계열 (#4CAF50, #11998e)
5. "다운로드" → PNG 형식

#### C. 추천 Canva 템플릿 스타일
- **미니멀 & 깔끔**: 파스텔 톤 + 큰 제목
- **전문가 느낌**: 다크 배경 + 흰색 텍스트
- **친근한 느낌**: 일러스트 + 부드러운 색상
- **카드뉴스 스타일**: 번호 + 포인트 요약

### 🎨 기타 이미지 제작 도구

#### 1. **미리캔버스** (한국형 Canva)
- URL: https://www.miricanvas.com
- 장점: 한글 폰트 많음, 한국 감성 템플릿
- 용도: 블로그 썸네일, 카드뉴스

#### 2. **Figma** (전문가용)
- URL: https://figma.com
- 장점: 세밀한 디자인 가능
- 단점: 학습 곡선

#### 3. **Unsplash / Pexels** (무료 이미지)
- [Unsplash](https://unsplash.com): 고품질 무료 사진
- [Pexels](https://pexels.com): 무료 사진 + 비디오
- 검색어: "meditation", "mental health", "peaceful", "nature"

#### 4. **AI 이미지 생성**
- **ChatGPT (DALL-E)**: 프롬프트로 이미지 생성
  - 예: "평화로운 숲, 햇살, 마음의 안정, 일러스트 스타일"
- **Midjourney**: 고품질 AI 이미지 (유료)
- **Stable Diffusion**: 무료 AI 이미지

## 5. 블로그 꾸미기 팁

### A. 헤더 이미지
- 크기: 1920 x 400px 권장
- 내용: 숲 이미지 + "숲울림 - 마음의 안식처" 텍스트
- Canva 템플릿: "YouTube Banner" 크기 활용

### B. 프로필 설정
- 프로필 사진: 나무/숲 아이콘 (Canva에서 제작)
- 소개: "24시간 AI 심리상담 서비스 숲울림의 공식 블로그입니다"

### C. 카테고리 설정
```
심리건강 기본
정신건강 질환
스트레스 관리
생활습관 개선
대인관계
직장생활
청소년 정신건강
```

### D. 글 서식 통일
```
제목: H1 (진한 초록색)
소제목: H2 (연한 초록색)
본문: 16px, 줄간격 180%
강조: 노란색 형광펜 or 굵게
```

## 6. 포스팅 스케줄 추천

- **월**: 정신건강이란 무엇인가요?
- **수**: 우울증 이해하기
- **금**: 스트레스 관리 5가지 방법

### Week 2
- **월**: 불안장애 극복하기
- **수**: 수면과 정신건강
- **금**: 대인관계 스트레스 해결법

### Week 3~8
- 매주 2-3개씩 꾸준히 포스팅
- 시간대: 오전 10시 or 저녁 8시 (트래픽 많은 시간)

## 7. SEO 최적화 (검색 노출)

### A. 제목 작성법
❌ 나쁜 예: "우울증"
✅ 좋은 예: "우울증 극복하는 5가지 방법 | 전문가 조언"

### B. 태그 활용
```
#심리상담 #우울증 #불안장애 #스트레스관리
#정신건강 #멘탈케어 #심리치료 #AI상담
#마음건강 #청소년상담 #직장인상담
```

### C. 본문 키워드 포함
- 첫 문단에 핵심 키워드 3번 이상
- 소제목에 키워드 포함
- 이미지 alt 텍스트에 키워드

### D. 내부 링크
- 다른 포스트 링크 걸기
- "이 글도 읽어보세요: [스트레스 관리 방법]"

## 10. 추천 콘텐츠 믹스

### 비율 추천:
- 70%: 교육 콘텐츠 (사이트 내용 재가공)
- 20%: 뉴스/이슈 (정신건강의 날, 자살예방주간 등)
- 10%: 개인 스토리/후기 ("AI 상담 체험기")

### 카드뉴스 예시:
1. "우울증 vs 우울감, 차이점은?"
2. "불안할 때 즉시 실천할 3가지"
3. "직장인 번아웃 자가진단"
